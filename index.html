Stuff and things

  • ascii-tarot

    this is probably the project I am most proud of, and I have continued to work on it over several years. I love ASCII art, and illustrated all of the cards using nano as my text editor.
    • ascii-tarot on npm
      the OG node package: a free command-line interface for tarot card reading you can install on your computer
    • irl edition
      a limited print run of 101 real card decks printed in black and green with a nice black velvet bag
      I used to have a Stripe payment link set up to take orders for them but due to dwindling supplies now only send them special order, email me to see if I still have a few (or if there are enough inquiries... to print another batch!)
    • discord bot
      discord is cute, I'm hip with it. you can add the ascii-tarot discord bot to your server and use the /draw command to draw a random card

  • MONTAG

    • MONTAG.WTF
      MONTAG Magazine was a project from the marketing department of Grover, the first startup I worked for in Berlin from March 2017 – April 2019. It had a pretty cool website that is now only available through the Internet Archive Wayback Machine. There were three print issues: TOGETHER TOMORROW, BETTER BODIES, and CODING CREATIVITY. There was also a short-lived (15 episode) podcast called the MONTAGE, which you can still listen to on Soundcloud.
    • MONTAG.XYZ
      When MONTAG the magazine ended in early 2019, I was beginning to be interested in AI text generation and created MONTAG.XYZ as "the magazine that writes itself". I took all of the past articles' text and trained an instance of GPT-2 with Max Woolf's gpt-2-simple and the now-defunct ML ops platform Spell ML (bought by Reddit in 2022)
